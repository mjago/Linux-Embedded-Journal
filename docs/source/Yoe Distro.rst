Yoe Distro
==========

Introduction
------------

The `Yoe Distribution <https://www.yoedistro.org/>`_ helps build and
maintain products using Embedded Linux. It has a simple directory
structure and tooling, and includes sane default policies and
templates, and utilizes a simple Docker container to simplify project
re-build over the product lifecycle. Yoe Distro uses
Yocto/OpenEmbedded in the background.
